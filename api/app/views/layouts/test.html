Additionally, we're aware some users have large repertoires stored as PGN files. We are happy to announce you can now import these into Chess HQ. All variations, comments, and arrows will automatically be added to your repertoire. To import a PGN, create a repertoire or access an existing repertoire, then click the "Import PGN" button in the area above the move list. This will be on the right side of your screen on a desktop or under the board on mobile. These fetures are free for all users.
